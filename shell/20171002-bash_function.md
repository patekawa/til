# Bashで関数を定義する

> 2017.10.02

---

## Bashの関数のフォーマット

### 関数の定義
```
function hello() {
    echo "Hello! $1\n"
}
```

### 関数の呼び出し
```
hello "foo"
```

## 引数と戻り値

### 引数
`$1`や`$2`で取得する．
bashの特殊変数が使える．
`$0`はbash変数名ではなくbashスクリプトの実行コマンドが入るので注意．

### 戻り値
`echo`で標準出力する．

Bashのグローバル変数でも指定できるが，環境が汚染されるので好ましくない．
